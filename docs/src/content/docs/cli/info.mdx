---
title: palettize info
description: Display information about Palettize with a colorful banner
---

The `info` command displays a colorful ASCII art banner and information about Palettize.

## Usage

```bash
palettize info
```

## Output

The command displays:

- **ASCII art banner** - A stunning and gloriously styled "palettize" logo
- **Version** - Current installed version
- **Description** - Brief description of the tool
- **GitHub URL** - Link to the source repository
- **Documentation URL** - Link to this documentation site
- **Random quote** - An inspirational quote about color from artists and thinkers

## Example

```bash
$ palettize info
```
